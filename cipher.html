Blockchain is a system that exists for security purposes.
It is a ledger that is unable to be changed
which records transactions and tracks assets.
It only allows permissioned network members
to access and see the full transaction history.
No one can change the ledger and if
there is an error then a new transaction
needs to be made which is then stored
along with the first transaction. So what actually
happens when a transaction is completed?
It is recorded as a ‘block’ of data
that contains everything from the transaction.
Each block is then attached to other blocks.
They form a chain and the longer it gets the
more immutable it becomes which removes the possibility
of hacking and changing the transaction history.
